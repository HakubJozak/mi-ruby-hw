# MI-RUB

## Requirements

   * ruby 2.0 or higer

     ruby -v
     
   * install `rspec` if you don't have it yet

     gem install rspec
 
## Usage

    
Then you can test your code with command (from folder that contains `spec` directory):

    rspec spec
    
or by specifing just a single file, e.g.:

    rspec spec/iterator_spec.rb
